# Tic-Tac-Toe
This project is a simple and interactive implementation of the classic Tic Tac Toe game. The game allows two players to take turns marking spaces on a 3Ã—3 grid using X and O symbols. The objective is to place three of the same symbols in a row, column, or diagonal to win the game.

The project is developed to demonstrate fundamental programming concepts such as loops, conditional statements, functions, and user input handling. It provides an easy-to-understand structure, making it suitable for beginners who want to learn game logic and basic software development.

Features:

Two-player gameplay

Win and draw detection

Simple and user-friendly interface

Efficient game logic implementation

Technologies Used:

( HTML, CSS, JavaScript)
